Getting started¶

You can use CardioMood API for getting user lists, their sessions and TR intervals.
To get access to API leave your request here.

Users list¶

Get list of users.

Resource URL¶

http://api.cardiomood.com/getUsers.php

HTTP method¶

GET

Parameters¶

http argument Type Options Description Example
token text (10 chars) required Authorizes to use the service 8bd3dc2C58

Example¶

GET¶

http://api.cardiomood.com/getUsers.php?token=TOKEN

Replace TOKEN with your token

Response¶

User session list¶

Get list of sessions of particular user or in certain time interval.

Resource URL¶

http://api.cardiomood.com/getSessions.php

HTTP method¶

GET

Parameters¶

http argument Type Options Description Example
token text (10 chars) required Authorizes to use the service 8bd3dc2C58
userId text (10 chars) optional Specifies the userId to process kYaNa09Gs1
start timestamp optional Specifies start time point of sessions 1425365365683
end timestamp optional Specifies end time point of sessions 1425365425706

Example¶

You need to specify at least one of optional query parameter:

GET¶

http://api.cardiomood.com/getSessions.php?token=TOKEN&userId=kYaNa09Gs1

Replace APIKEY with your key 



                            


                            
                            
GET¶
                            


                            

                            
http://api.cardiomood.com/getSessions.php?token=TOKEN&start=1425365365683
